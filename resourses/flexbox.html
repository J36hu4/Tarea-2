Flexbox

Flex Container


  1. flex-direction: Define la dirección en la que los elementos flexibles se colocan en el contenedor

    • row: Los elementos se colocan en una fila horizontal (de izquierda a derecha).
    • row-reverse: Los elementos se colocan en una fila horizontal pero en orden inverso (de derecha a izquierda).
    • column: Los elementos se colocan en una columna vertical (de arriba hacia abajo).
    • column-reverse: Los elementos se colocan en una columna vertical pero en orden inverso (de abajo hacia arriba).

  2. flex-wrap: Permite que los elementos flexibles se envuelvan en múltiples líneas.

    • nowrap: Todos los elementos se colocan en una sola línea (valor por defecto)
    • wrap: Los elementos se envuelven en múltiples líneas si es necesario.
    • wrap-reverse: Los elementos se envuelven en múltiples líneas pero en orden inverso.

  3. flex-flow: Es una abreviatura para flex-direction y flex-wrap

    • Ejemplo: flex-flow: row wrap; (equivalente a flex-direction: row; flex-wrap: wrap;).

  4. :

    • flex-start: Los elementos se alinean al inicio del contenedor.
    • flex-end: Los elementos se alinean al final del contenedor.
    • center: Los elementos se alinean en el centro del contenedor.
    • space-between: Los elementos se distribuyen uniformemente con el primer elemento al inicio y el último al final.
    • space-around: Los elementos se distribuyen uniformemente con espacio igual alrededor de cada elemento.
    • space-evenly: Los elementos se distribuyen uniformemente con espacio igual entre ellos.

  5. align-items: Alinea los elementos flexibles a lo largo del eje transversal.

    • flex-start: Los elementos se alinean al inicio del eje transversal.
    • flex-end: Los elementos se alinean al final del eje transversal.
    • center: Los elementos se alinean en el centro del eje transversal.
    • baseline: Los elementos se alinean según la línea base de su contenido.
    • stretch: Los elementos se estiran para llenar el contenedor (valor por defecto).

  6. align-content: Alinea las líneas del contenedor flex cuando hay espacio extra en el eje transversal.

    • flex-start: Las líneas se alinean al inicio del contenedor.
    • flex-end: Las líneas se alinean al final del contenedor.
    • center: Las líneas se alinean en el centro del contenedor.
    • space-between: Las líneas se distribuyen uniformemente con el primer línea al inicio y la última al final.
    • space-around: Las líneas se distribuyen uniformemente con espacio igual alrededor de cada línea.
    • stretch: Las líneas se estiran para llenar el contenedor (valor por defecto).

Flex Items


  1. order: Controla el orden de los elementos flexibles.

    • Valor: Número entero (por defecto es 0). Un número menor se coloca antes y un número mayor se coloca después.

  2. flex-grow: Define la capacidad de un elemento para crecer si es necesario.

    • Valor: Número (por defecto es 0). Un valor mayor permite que el elemento crezca más en comparación con otros elementos.

  3. flex-shrink: Define la capacidad de un elemento para encogerse si es necesario.

    • Valor: Número (por defecto es 1). Un valor mayor permite que el elemento se encoja más en comparación con otros elementos.

  4. flex-basis: Define el tamaño base de un elemento antes de que se distribuya el espacio restante.

    • Valor: Tamaño (por defecto es auto). Puede ser un valor absoluto (px, em, etc.) o relativo (%).

  5. flex: Es una abreviatura para flex-grow, flex-shrink y flex-basis.

    • Ejemplo: flex: 1 1 auto; (equivalente a flex-grow: 1; flex-shrink: 1; flex-basis: auto;).

  6. align-self: Permite alinear un elemento individualmente a lo largo del eje transversal.

    • Valores: auto (valor por defecto, hereda de align-items), flex-start, flex-end, center, baseline, stretch.

Flex Responsive


  1. Para lograr un diseño responsivo con Flexbox, utilizamos media queries junto con las propiedades de Flexbox.
    • Ejemplo: 
      •     .container {
        display: flex;
        flex-wrap: wrap;
        justify-content: space-between;
    }

    .item {
        flex: 1 1 200px;
    }

    @media (max-width: 600px) {
        .container {
            flex-direction: column;
        }
    }
