Box Model

Componentes del Box Model o Modelo de Caja


1. Contenido (Content): Es el área donde se muestra el contenido, como texto o imágenes.
2. Relleno (Padding): Es el espacio entre el contenido y el borde. El relleno es transparente.
3. Borde (Border): Es el borde que rodea el contenido y el relleno. Puede tener diferentes estilos, grosores y colores.
4. Margen (Margin): Es el espacio exterior al borde. El margen también es transparente y se utiliza para separar los elementos entre sí.

En la ilustracion anterior puede ver un contedor con un ancho(width) de 150px y con un alto(height) 150px. Ademas, tiene un margen de 15px por cada lado, un borde de 10px y un Padding de 10px.
Si sumamos cada lado podremos ver que tiene un ancho de 220px, veamos porque:

Content: 150px
Paddin izquido: 10px
Paddin derecho: 10px
Border izquierdo: 10px
Border derecho: 10px
Margin izquido: 15px
Margin derecho: 15px
Total: 150px + 10px + 10px + 10px + 10px + 15px + 15px = 220px
Así mismo pasa con la altura

Elementos en Bloque


Los elementos de bloque ocupan todo el ancho disponible de su contenedor y siempre comienzan en una nueva línea. Algunos ejemplos comunes incluyen:
- div
- p
- h1 a h6
- ul, ol, li
- table, tr, td


Elementos en Linea


Los elementos en línea solo ocupan el ancho necesario para su contenido y no comienzan en una nueva línea. Algunos ejemplos comunes incluyen:
- a
- span
- img
- strong
- em
