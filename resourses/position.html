Propiedad Position


La propiedad position en CSS se utiliza para especificar el método de posicionamiento de un elemento. Los valores principales son static, relative, absolute, fixed y sticky.

1. Static: Es el valor por defecto. Los elementos se posicionan según el flujo normal del documento.

2. Relative: El elemento se posiciona relativo a su posición normal. Puedes usar top, right, bottom, y left para ajustar su posición.

3. Absolute: El elemento se posiciona relativo a su contenedor más cercano que no tenga position: static. Si no hay tal contenedor, se posiciona relativo al html.

4. Fixed: El elemento se posiciona relativo a la ventana del navegador y no se mueve al hacer scroll.

5. Sticky: Es una combinación de los valores relative y fixed. Un elemento con position: sticky se comporta como un elemento relative hasta que alcanza un umbral definido por las propiedades top, right, bottom, o left. Una vez que alcanza ese umbral, se comporta como un elemento fixed.


Propiedad Float y Clear


La propiedad float se utiliza para hacer que un elemento flote a la izquierda o derecha de su contenedor, permitiendo que el texto y otros elementos fluyan a su alrededor. La propiedad clear se utiliza para controlar el comportamiento de los elementos flotantes

1. Float: Permite que los elementos floten a la izquierda o derecha.

2. Clear: Evita que los elementos floten a los lados especificados.


Propiedad Display


La propiedad display controla cómo se muestra un elemento en la página.

1. Block: El elemento se comporta como un bloque.

2. Inline: El elemento se comporta como un elemento en línea.

3. Inline-Block: El elemento se comporta como un bloque en línea.

4. None: El elemento no se muestra.


Propiedad Overflow


La propiedad overflow controla qué sucede cuando el contenido de un elemento es más grande que su área.

1. Visible: El contenido desbordado es visible.

2. Hidden: El contenido desbordado es oculto.

3. Scroll: Se añaden barras de desplazamiento para ver el contenido desbordado.

4. Auto: Se añaden barras de desplazamiento solo si es necesario.


Propiedad Visibility


La propiedad visibility controla la visibilidad de un elemento.

1. Visible: El elemento es visible.

2. Hidden: El elemento es invisible pero ocupa espacio.

3. Collapse: Similar a hidden, pero para elementos de tabla


Propiedad Z-index
